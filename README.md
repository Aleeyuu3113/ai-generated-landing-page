# FlowNest Landing Page

A simple responsive landing page for a remote team planning tool.

## What’s included

- `index.html` — semantic HTML structure with header, hero, features, CTA, and footer.
- `style.css` — responsive styling using CSS Grid and Flexbox with mobile media queries.

## Notes

- AI helped suggest the app name, hero headline, feature copy, and colour palette.
- I wrote all of the HTML and CSS by hand to implement the page structure and responsive layout.

## Preview

Open `index.html` in a browser or run a local server from this folder.
