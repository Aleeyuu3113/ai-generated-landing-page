# AI-Generated Landing Page

A responsive one-page marketing website for a productivity app called **FlowTime**.

## What is included

- `index.html` — semantic HTML5 structure with hero, features, CTA, and footer sections
- `style.css` — modern styling using Flexbox/Grid and responsive media queries

## AI assistance

AI helped with:

- choosing the app name and product pitch
- drafting hero text, feature descriptions, and CTA copy
- selecting a clean color palette and layout concept

All HTML and CSS code was written manually to meet the assignment requirements.

## How to preview

Open `index.html` in a browser or use Live Server in VS Code.

## Notes

The page is responsive and includes a media query for smaller screens.
